# Launch Checklist

## Domain and Infrastructure

- [ ] Domain connected.
- [ ] SSL active.
- [ ] Production environment variables configured.
- [ ] `/api/health` returns OK.
- [ ] Error monitoring connected.
- [ ] Analytics connected.
- [ ] Backup enabled and restore tested.

## Storefront

- [ ] Homepage CMS content reviewed.
- [ ] Product listing and filters tested.
- [ ] Search tested for popular and no-result terms.
- [ ] Product detail pages reviewed on mobile and desktop.
- [ ] Sticky mobile add-to-cart verified.
- [ ] Cart update/remove tested.
- [ ] Checkout tested as guest and logged-in customer.
- [ ] Order success and failure pages tested.

## Catalog and Compliance

- [ ] Product data verified.
- [ ] Prices, MRP, discount, serving count, and protein per serving verified.
- [ ] Label images uploaded.
- [ ] Nutrition facts verified.
- [ ] Ingredients and allergen declarations verified.
- [ ] Warning text and “Not for medicinal use” visible.
- [ ] Lab report/COA links checked where available.
- [ ] Disease cure claims reviewed and removed.

## Inventory

- [ ] Warehouses verified.
- [ ] Batch numbers verified.
- [ ] Expiry rules checked.
- [ ] FEFO reservation tested.
- [ ] Low-stock thresholds checked.
- [ ] Expiry alerts checked.
- [ ] Stock adjustment audit logs checked.

## Payments and Shipping

- [ ] Payment tested.
- [ ] COD tested.
- [ ] Refund test completed.
- [ ] Shipping tested.
- [ ] Tracking number sync tested.
- [ ] Return/RTO flow tested.

## Admin and Security

- [ ] Admin login secure.
- [ ] Mock admin login disabled.
- [ ] Role permissions reviewed.
- [ ] 2FA provider selected or enabled.
- [ ] Session timeout enabled.
- [ ] Rate limiting enabled.
- [ ] Audit logs reviewed.

## Legal and Trust Pages

- [ ] Return policy published.
- [ ] Shipping policy published.
- [ ] Privacy policy published.
- [ ] Terms published.
- [ ] Contact page working.
- [ ] FAQ reviewed.

## SEO and Growth

- [ ] Metadata reviewed.
- [ ] Sitemap submitted.
- [ ] Robots.txt reviewed.
- [ ] Open Graph image verified.
- [ ] Product schema spot-checked.
- [ ] Email/SMS/WhatsApp templates tested.
- [ ] Coupon abuse prevention rules reviewed.

## Launch Decision

- [ ] Go-live approved by business owner.
- [ ] Rollback plan documented.
- [ ] Support contact and escalation owner assigned.
